# 🧠 Quiz App

A fun and interactive General Knowledge Quiz App built with pure HTML, CSS, and JavaScript.

## 🔗 Live Demo
[Play the Quiz!](https://rohinipriyavasu.github.io/quiz-app)

## ✨ Features
- 10 categories to choose from
- 3 difficulty levels (Easy, Medium, Hard)
- Real-time questions from Open Trivia DB API
- Green/Red answer feedback
- Score tracking
- Fully responsive design

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Open Trivia DB API

## 🚀 How to Run Locally
1. Clone the repo
   git clone https://github.com/ROHINIPRIYAVASU/quiz-app.git
2. Open index.html in your browser
